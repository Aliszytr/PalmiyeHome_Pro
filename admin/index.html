📋 Rezervasyon Yönetimi
Takvim verilerini düzenlemek için data/bookings.js dosyasını cPanel File Manager veya FTP ile düzenleyin.

Format: Tarihler "YYYY-AA-GG" formatında olmalıdır.
Aşağıda mevcut veriyi görüntüleyebilirsiniz.
Palmiye Home Takvim Yönetimi
Düzenleme Nasıl Yapılır:
1. cPanel → File Manager → data/bookings.js dosyasını açın
2. Rezervasyon tarihlerini ekleyin veya kaldırın
3. Dosyayı kaydedin

Örnek format:
{ "from": "2026-06-01", "to": "2026-06-06" }
